# Source Log

## Purpose

This file tracks the types of sources used in this case study, what each source was intended to support, and how each source should be interpreted.

Because this project relied on public signals and product-flow observation rather than internal platform data, source discipline matters.

## Source Categories

### 1. Public visibility signals

These sources were used to support the observation that Spotify Wrapped appeared to generate stronger visible public circulation than Apple Replay.

Examples may include:
- public social posting presence
- recurring seasonal visibility
- public search activity
- visible sharing behavior across platforms

Use: directional support for the existence of a visible distribution gap
Do not use: exact share volume or exact total reach claims

### 2. Product-flow observations

These sources were used to compare how each experience appeared to move users from private consumption to public sharing.

Examples may include:
- screenshots of recap flow
- access path observations
- number of user steps
- visibility of share prompts
- final output readiness for posting

Use: comparative analysis of friction, share readiness, and broadcast flow
Do not use: definitive proof of user intent or exact conversion rates

### 3. Supporting reference material

These sources were used to strengthen context, not to serve as primary proof.

Examples may include:
- public articles
- commentary about annual recap behavior
- platform descriptions
- screenshots used for visual reference

Use: contextual support and framing  
Do not use: direct proof of exact platform-level performance

## Evidence Strength Guide

### Stronger support
- directly observed product-flow differences
- repeated visible sharing patterns
- recurring public circulation signals

### Moderate support
- directional public search interest
- consistent public commentary or discussion themes
- repeated visual evidence from multiple public sources

### Weaker support
- single screenshots without broader context
- isolated commentary
- unsupported numerical extrapolation
